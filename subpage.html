Guide to Linux Commands

Command Description How It Can Be Used
cd Takes the user to the directory specified. If placed alone, it takes the user to the home directory cd scripts
rm Removes (delete) the file specified rm file
cp Copies one file to another file cp file1 file2
pwd Prints the current directory you are in pwd
ls Tells what files are in the current directory ls
mkdir Creates a directory mkdir folder
man Gives a detailed summary of all the capabilites of a command man cd
mv Renames a file mv file Hello
locate Locates a file in the LInux system. locate helloworld
echo Repeat the message that is made after the command. echo hello
